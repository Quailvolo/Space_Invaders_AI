# Space Invaders AI Edition

## Опис гри
Це класична аркадна гра, де гравець керує космічним кораблем і захищає Землю від хвиль прибульців.

## Як запустити
1. Завантажте файл `space-invaders.html`.
2. Відкрийте його у будь-якому сучасному браузері (Chrome, Safari, Firefox).

## Керування
- **A / D** або **Стрілки**: Рух кораблем
- **Пробіл**: Стрільба
- **Enter**: Рестарт гри

## AI Agent
Для створення гри використовувався **Gemini (Google)**.

## Процес створення
1. На основі наданого файлу `AGENTS.md` було сформовано промпт для генерації коду.
2. AI згенерував структуру HTML5 Canvas та логіку на Vanilla JavaScript.
3. Було додано звукові ефекти за допомогою Web Audio API для ретро-атмосфери.
4. Додано систему збереження рекордів у LocalStorage.
